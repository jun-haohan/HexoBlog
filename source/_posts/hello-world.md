## 使用指南

### 创建笔记
``` bash
$ hexo new "My New Post"
```

### 本地启动
``` bash
$ hexo s
```

### 生成静态文件
``` bash
$ hexo g
```

### 远端部署
``` bash
$ hexo d
```
