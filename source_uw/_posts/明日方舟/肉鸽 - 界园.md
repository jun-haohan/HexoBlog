# 结局
1. 一结局：不触发任何选项。

推荐星熊堵boss门，左边先锋守，右边电弧守，输出帮忙打boss

2. 二结局：树洞→抉择→一般选30源石锭选项拿【隐蝶墨】→在后续作战打掉岁脚→自动获得【墨化残碑】。

    左路EW+伊桑/阿斯卡纶，中路阿能单守，右路银灰+奶，夜刀打野，最好在有一个快活保底。建议先清完小怪再打boss，星熊/初雪/祥子输出。
    左路17个怪，不够胡请直接放生。
    过于胡时请注意，boss半血强制全场撤退，请准备好决战技抗线

    (注意：源石锭不足30选第二个，但有概率跑结局；如果有藏品【寅诗壶】且确保你能很大概率投出花钱，可以选寅诗壶选项，但未投出花钱仍会跑结局)
3. 三结局：树洞→特殊杂疑(一般是树洞点亮的第一个）→五伺烛客选项→拿【忘生珍珑】→指点迷津→选【持有忘生珍珑，遭遇特殊战斗】→打过拿藏品【云与漆】

推荐萌萌香+星熊+ins+两个高台输出(EW/阿能/酒神等)挂机，最好有夜刀补漏

4. 四结局：树洞→易与(一般是你点进去的第一个)→花五血拿【无封长盒】→找先行一步节点入卷一名干员→拿藏品【无墨长卷】

左路萌萌香+星熊可守（不够胡加电弧），角峰抗小boss，二阶段右路EW输出，决战技多多益善。

5. 五结局：树洞→故肆→使故事走向不同结局→拿【追忆仪】(注意：会清空你所有源石锭和希望，而且会卡你发育）→误入奇境→持有【追忆仪】选项→通关拿藏品【小磨唧】

    电弧 星熊 初雪 银灰 夜刀 酒神 投锋+先锋 可对空输出

    (五结局超大蛋：不息：指点迷津→拆解岁护→获得【不息】与九票券；不赦：持有【小磨唧】→树洞→故肆→持有【小磨唧】选项，拿【不赦】)
6. 补充：一二结局冲突，三四结局冲突。（二比一难，三比四难）

# 干员推荐：
1. 先锋：银灰
2. 近卫：祥子，玛恩纳
3. 重装：星熊
4. 狙击：EW，提丰
5. 术师：初雪，霍尔海雅
6. 医疗：乌啾
7. 辅助：电弧，灵知，酒神
8. 特种：夜刀，阿能，诗怀雅，望

# 天师府分队策略：
1. 开局：优先选择2券，电弧 + 苏苏洛 + 骋风，一层追紧急不拐弯，第一张券直接用。

    先锋抓银灰，重装抓星熊，特种抓阿能，辅助精二电弧。 近卫，狙击，术师，医疗抓四星。
2. 一层商店一定要燃烛电弧。若剩余16块及以上且需要精二，可考虑刷新。（记得留2源石锭）
3. 一层进洞，卡三结局优先级（三结局暂时不会点，因此先开分支多的路，防止大片区域被三结局杂疑堵死），传说尽量在血少的时候走。
   一层尽量不考虑降档通过。

   战斗节点：精一电弧可过震坎2，精二电弧可过贪妄2、愠怒3 。
4. 二层指点迷津：渡劫。星熊初雪银灰骋风大概率成功，否则考虑电弧以保证奖励。
5. 二层根据收益决定拐几次，优先紧急、先行一步、安全的角落。
6. 二层进洞尽量打祸乱，最后拿长盒。

# 通用策略
1. 开局：拿热水壶，电弧+阿能+桃子开局，一层三站攒券，不进洞
2. 二层指点迷津点人，进洞尽可能打（传说节点要在走五个节点之后再进）
3. 指点迷津都点人，除非没有券或者队伍齐了，拿来渡劫
